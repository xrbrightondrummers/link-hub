REQUEST US TO PLAY

Although we are an Extinction Rebellion group and prioritise climate-based activism, we often join forces with and play in support of other actions and causes that align with our own ethics and beliefs, such as anti-racism, anti-fascism, and anti-war events. If your movement or action is based on care, empathy, and the wellbeing of all people, in particular marginalised people, we're probably on the same page. We're particularly interested in forging stronger connections with local Brighton groups.

If you'd like to request our support at an action or event, the best way to get in touch is to message us on Instagram, Facebook, or Telegram. Please bear in mind we need as much notice as possible, ideally at least 2 weeks. Every band member is a volunteer, we all have busy schedules and many demands on our time, and it takes time to decide and organise to all turn up at any given event. When getting in touch to request us, please bear in mind we'll need the following information:

Information required to request us

- Name of the event

- Description and purpose of the event

- Date

- Start time.

- Duration. Expected number of hours, can be approximate - we will discuss.

- Location. If it's a march, please give start and end locations, route plan

- Accessibility information about the location or route

- A link to more information, eg. a web page, facebook or instagram account, or other resource where we can read about the full event details

Please message us on any of the following to get in touch:

Instagram Facebook Telegram
